A Tool to Remove White Space of `.pdf` Files.

**This project is under construction.**

## Usage
* Install [Framework 4.6.1],
* Run `PDF.Cropper\bin\Debug\PDF.Cropper.exe`.

## Built with
* Visual Basic .NET,
* Visual Studio Community 2015.
